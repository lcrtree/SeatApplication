# SeatApplication

#### 介绍

模块化设计的自动排座位，支持自定义规则和特效（暂时只制作了cli）

#### 画饼区
1. [] 内核部分制作
2. [] CLI部分制作
3. [] GUI部分制作完善

#### 编译安装教程

1.  安装virtualenv

```bash
python -m pip install virtualenv
```

2.  安装环境

```bash
python -m pip install requirements.txt -r
```

3.  开始使用！

#### 使用说明

1.  下载发布版后先解压！
2.  相关的配置文件在根目录
